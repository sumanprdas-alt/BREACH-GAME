# BREACH

**THREE OPERATIVES. NO BACKUP. BREAK THROUGH EVERYTHING.**

A Contra-style 2D run-and-gun platformer set in 2089. Built as a single HTML file — no dependencies, no build step. Just open in a browser and play.

## Play

Open `index.html` in any modern web browser. That's it.

## Story

**Year 2089.** A paramilitary syndicate called **Crimson Dawn** has seized Archipelago Sigma and activated **Project Starkfire** — a superweapon that sends a single plasma pulse to kill every power grid on Earth. Three operatives are deployed. No backup. No extraction plan.

## Controls

| Key | Action |
|-----|--------|
| ← → | Move / Aim |
| ↑ | Aim Up |
| ↓ | Prone |
| SPACE | Jump |
| X | Shoot |
| C | Grenade |
| ↓ + SPACE | Drop Through Platform |
| ENTER | Advance / Confirm |
